# renku-commons
Common utilities for Renku services

## Development
Building is done using [sbt](http://www.scala-sbt.org/).

To build the project:
```bash
$ sbt compile
[...]

[success] Total time: 19 s, completed 12-Sep-2017 09:29:15
```
